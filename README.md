# LichSOMA's Daily Calendar

> **이 모듈은 생성형 AI의 도움으로 제작되었습니다.**

## 호환성

- **Foundry VTT**: v14 이상
- **언어 지원**: 한국어
- **권장 모듈**: [`lichsoma-taskbar`](https://github.com/LichSOMA/lichsoma-taskbar) (있으면 작업표시줄에 날짜 표시 자동 연동)

## 소개

Foundry VTT용 **시간대·일자** 캘린더 모듈입니다. 월드의 시간을 **오전 / 오후 / 심야**로 구분하고, 필요하다면 **일자**를 추가해 관리할 수 있습니다.

## 주요 기능

### 시간대 표시

- 오전 / 오후 / 심야를 색상 오버레이로 화면에 반영합니다.
- 시간대를 바꾸면 부드러운 색상 오버레이(노을/밤)가 적용됩니다.

### 일자 카운트

- `1일차`부터 자유롭게 진행하고, 기준 날짜에 더해 실제 날짜로 환산해 표시합니다.

### 컨트롤 패널 (GM)

- 우측 하단의 모래시계 아이콘을 클릭하거나, 씬 컨트롤(조명) 도구의 모래시계 버튼을 클릭하면 패널이 열립니다.
- **오전 / 오후 / 심야** 버튼으로 시간대를 직접 지정합니다.
- **◀ / ▶** 화살표로 시간대를 한 단계씩 이동합니다. (캘린더 활성 시 자정을 넘기면 다음/이전 날짜로 이어집니다.)
- **« / »** 화살표로 일자(`N일차`)를 직접 증감합니다.

### 캘린더 위젯

- 좌측 상단에 월드 시간을 시각적으로 표시합니다.

### 모듈 설정

- **캘린더 기능 활성화**: 끄면 좌측 상단 위젯과 테스크바 날짜 표시가 사라지고, 시간대 컨트롤만 남습니다.
- **캘린더 기준 날짜 (YYYY/MM/DD)**: `1일차`에 해당하는 실제 날짜를 지정합니다. (기본값: `2000/01/01`) 예: 기준을 `2000/01/01`로 두고 `5일차`이면 `2000/01/05`로 환산되어 표시됩니다.
- **캘린더 폰트**: 현재 브라우저에 로드된 폰트(다른 폰트 모듈이 등록한 폰트 포함) 목록에서 선택할 수 있습니다.

### 테스크바 연동 (선택)

- [`lichsoma-taskbar`](https://github.com/LichSOMA/lichsoma-taskbar) 모듈이 활성화되어 있으면 작업표시줄 우측에 토글 버튼과 날짜가 표시됩니다.

## 설치

이 모듈은 공식 릴리즈를 통해 배포되지 않습니다. GitHub 저장소의 manifest URL로 직접 설치하거나, 모듈 폴더를 `Data/modules` 아래에 두고 Foundry VTT 모듈 관리에서 **LichSOMA's Daily Calendar**를 활성화합니다.
