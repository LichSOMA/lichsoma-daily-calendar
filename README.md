# LichSOMA's Daily Calendar

> **이 모듈은 생성형 AI의 도움으로 제작되었습니다.**

Foundry VTT용 시간대/일자 캘린더 모듈입니다. 월드의 시간을 **오전 / 오후 / 심야**로 구분하고, 필요하다면 **일자**를 추가해 관리할 수 있습니다.

## 제작 안내

이 모듈은 생성형 AI(LLM)의 도움을 받아 제작되었습니다. 코드 설계와 작성, 문서화 일부에 AI를 활용했으며, 모든 결과물은 제작자가 검토·수정한 뒤 배포되었습니다.

## 핵심 기능

- **시간대 표시**: 오전 / 오후 / 심야를 색상 오버레이로 화면에 반영
- **일자 카운트**: `1일차`부터 자유롭게 진행, 기준 날짜에 더해 실제 날짜로 환산
- **좌측 상단 캘린더 위젯**: 월드 시간을 시각적으로 한눈에 확인
- **우측 하단 컨트롤 패널**: GM이 시간/일자를 빠르게 조작
- **테스크바 연동** *(선택)*: [`lichsoma-taskbar`](https://github.com/LichSOMA/lichsoma-taskbar) 모듈이 활성화되어 있으면 작업표시줄 우측에 토글 버튼과 날짜 표시
- **폰트 선택**: FVTT의 커스텀 폰트 중에서 캘린더에 사용할 폰트를 자유롭게 선택

## 설치 방법

이 모듈은 공식 릴리즈를 통해 배포되지 않습니다. GitHub 저장소의 manifest URL로 직접 설치해 주세요.

1. Foundry VTT 셋업 화면에서 **모듈 관리 > 모듈 설치 > Manifest URL** 입력란에 저장소의 `module.json` 직접 링크를 붙여넣기
2. 설치 후 월드에서 모듈 활성화

## 기본 사용법

### 시간대 / 일자 변경 (GM)

1. 우측 하단의 모래시계 아이콘을 클릭하거나, 씬 컨트롤(조명) 도구의 모래시계 버튼을 클릭하면 컨트롤 패널이 열립니다.
2. 패널에서 다음을 조작할 수 있습니다.
   - **오전 / 오후 / 심야** 버튼으로 시간대 직접 지정
   - **◀ / ▶** 화살표로 시간대를 한 단계씩 이동 (캘린더 활성 시 자정을 넘기면 다음/이전 날짜로 이어짐)
   - **« / »** 화살표로 일자(`N일차`)를 직접 증감
3. 시간대를 바꾸면 화면에 부드러운 색상 오버레이(노을/밤)가 적용됩니다.

### 캘린더 켜기 / 끄기

- 모듈 설정 > **캘린더 기능 활성화**: 끄면 좌측 상단 위젯과 테스크바 날짜 표시가 사라지고, 시간대 컨트롤만 남습니다.

### 기준 날짜 설정

- 모듈 설정 > **캘린더 기준 날짜 (YYYY/MM/DD)**: `1일차`에 해당하는 실제 날짜를 지정합니다. (기본값: `2000/01/01`)
- 예: 기준을 `2000/01/01`로 두고 `5일차`이면 `2000/01/05`로 환산되어 표시됩니다.

### 폰트 변경

- 모듈 설정 > **캘린더 폰트**: 현재 브라우저에 로드된 폰트(다른 폰트 모듈이 등록한 폰트 포함) 목록에서 선택할 수 있습니다.
